Assessment & evaluation

Built for continuous improvement

SPECIAL uses mixed methods to assess progress toward partnership expansion, infrastructure strengthening, and community/industry engagement.

A modern, accessible layout for partners, faculty, students, and community stakeholders.

Formative evaluation

Ongoing data collection supports course-correction and refinement during the award period.

Summative evaluation

End-of-award report summarizes outcomes, strategies that worked, lessons learned, and overall impact.

Data sources

Implementation materials (minutes, participant lists, affiliations), program metrics, surveys, and partner/faculty/student perspectives where applicable.

Key metrics

  • Number of external partners
  • Number of collaborative projects
  • Number of participants and events
  • Participant survey feedback
